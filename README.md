# 이미지 비교 실험

## 설치 방법

1. 가상환경 생성 및 활성화

```bash
python -m venv venv
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate  # Windows
```

2. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
python src/test.py
```

## 필요 사항

- Python 3.6 이상
- 필요한 패키지들은 requirements.txt에 명시되어 있습니다.
